Creating Objects

To create a new object in Kubernetes, you must provide a "spec" that describes its desired state. We'll be using YAML files for this.
Example:

apiVersion: apps/v1
kind: Deployment
metadata:
name: devopstree-react
spec:
replicas: 1
template:
metadata:
labels:
app: devopstree-react
spec:
containers:
- name: devopstree-react
- image: vinaydhegde/devopstree-react:latest

    Required Fields:

  1. apiVersion - Kubernetes API version
  2. kind - the type of object you want to create
  3. metadata - info about the object so that it can be uniquely identified
