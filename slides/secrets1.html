Secrets

Secrets are used to hold sensitive data such as passwords, API tokens, or SSH keys.


k8s/secret.yml:


apiVersion: v1
kind: Secret
metadata:
  name: postgres-credentials
type: Opaque
data:
  user: c2FtcGxl
  password: cGxlYXNlY2hhbmdlbWU=

The user and password fields are base64 encoded strings (security via obscurity):


$ echo -n "" | base64

Add the secrets:


$ kubectl apply -f ./kubernetes/secret.yml
