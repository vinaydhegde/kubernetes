Minikube

Minikube is a tool which allows developers to use and run a Kubernetes cluster locally.

Getting started
First, install the following tools:

  1. Install a Hypervisor (like VirtualBox or HyperKit) to manage virtual machines
  2. Install and Set Up kubectl to deploy and manage apps on Kubernetes
  3. Install Minikube
Then, start the cluster:
minikube start --vm-driver=virtualbox minikube dashboard
